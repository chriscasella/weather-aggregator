<h1> Weather Aggregator </h1>
<br>
<br>
<h3>What it does</h3>
===
<p>This app pulls from weather from a few sources and displays them as cards to be viewed in parallel to each other. Different
weather sources say different things at times and it would be nice to know what the general view is.</p>

<h3> Technologies Used </h3>

<ul>
    <li>AngularJS 1.6.5</li>
    <li>BootStrap 4</li>
    <li>FontAwesome 5</li>
    <li>Material Design AngularJS</li>
    <li>DarkSky API</li>
    <li>National Weather Service API</li>
    <li>Wunderground API</li>
    <li>Weatherbit.io API</li>
</ul>
